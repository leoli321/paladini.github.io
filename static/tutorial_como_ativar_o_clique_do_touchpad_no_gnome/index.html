[Tutorial] Como ativar o clique do touchpad no GNOME

07 August 2014 on Software Livre, Tutoriais, Fedora, Ubuntu

O Gnome é sem sombra de dúvidas uma das interfaces gráficas mais utilizadas nas distribuições Linux. Um pequeno "problema" que pode atrapalhar alguns usuários de notebook que usam o Gnome é uma configuração que bloqueia o uso do clique do touchpad.

Essa função, de grande importância para muita gente, já vem desabilitada por padrão na interface Gnome. Se você deseja ativar o "click" no touchpad, esse tutorial é para você. 

O Tutorial
Passo 1 - Abra a sua tela de configurações: Passo 2 - Clique em "Mouse & Touchpad", então uma nova janela será aberta:   Passo 3 - Na caixinha onde está escrito "Tap to click", clique para deixá-la marcada / selecionada: E pronto, o seu touchpad agora já tem a função de clique! 

Se tiver alguma dúvida, sugestão ou crítica, fique a vontade para utilizar a seção de comentários logo abaixo, cada uma será lida!
