[Tutorial] Rails 4 - Como renomear Rotas REST da URL

08 December 2013 on Tutoriais, Rails

Recentemente criei um novo projeto em Rails 4 e estou quebrando um pouco a cabeça com as grandes mudanças sofridas neste fantástico framework. 

Alguma vez você já deve ter criado algum modelo com determinado nome e se arrependeu disso depois, pois a URL não condizia com o conteúdo das páginas. Imagine você tendo um agregador de links, faria sentido ter uma URL como a seguinte?

http://www.meusite.com/items/1

Provavelmente não, mas felizmente o Rails fornece uma grande opção para você renomear as suas rotas REST da URL. Nesse caso, no seu arquivo routes.rb teria algo como:

resources :items

Caso você queria renomear todas as suas Rotas para algo da sua escolha é muito fácil:

resources :items, :path => "links"

Agora teremos uma nova URL para a mesma Rota:

http://www.meusite.com/links/1

Muito simples, não? Esta postagem foi baseada nesta pergunta do Stack Overflow
