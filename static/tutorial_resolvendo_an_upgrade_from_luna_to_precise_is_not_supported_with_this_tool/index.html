[Tutorial] Resolvendo 'An upgrade from 'luna' to 'precise' is not supported with this tool.'

27 October 2013 on Software Livre, Tutoriais

Se você é usuário do novíssimo Elementary OS

Neste pequeno tutorial vou mostrar como resolver um erro que me perturbou por muito tempo:

An upgrade from 'luna' to 'precise' is not supported with this tool.

Esse erro acontecia toda vez que eu ia atualizar o sistema e seus pacotes através do "Update Manager", o que me deixava muito irritado. Abaixo são os erros que eu recebia:
O seu problema é o mesmo que acima, ou ao menos similar? Se for, vamos resolver!
1º Passo - *Abra o seu Terminal apertando "Ctrl + Alt + T".2º Passo - *Digite os seguintes comandos, na ordem que estão:

  
sudo apt-get update 

sudo apt-get dist-upgrade

Pronto, agora o seu problema provavelmente deve ter sido resolvido! Se você ainda continua com o erro "An upgrade from 'luna' to 'precise' is not supported with this tool." ou tem outro, poste nos comentários para que eu possa ajudá-lo.
