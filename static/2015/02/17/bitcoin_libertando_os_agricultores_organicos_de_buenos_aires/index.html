Bitcoin: libertando os agricultores orgânicos de Buenos Aires

17 February 2015 on Software Livre, Projetos, Bitcoin

O Bitcoin trouxe grandes novidades para o sistema monetário internacional, que já vinha caminhando de forma quase estática desde o século passado. Com as novidades tecnológicas introduzidas pelo Bitcoin é possível realizar pagamentos em qualquer horário do dia para qualquer lugar do planeta com taxas que beiram os 5 centavos de dólar (US$0,05), algo jamais alcançado pelo sistema financeiro atual.
