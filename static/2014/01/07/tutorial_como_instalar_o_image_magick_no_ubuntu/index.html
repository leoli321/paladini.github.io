[Tutorial] Como instalar o Image Magick no Ubuntu

07 January 2014 on Tutoriais

Tenho trabalhado em um projeto open-source chamado , o primeiro diretório Bitcoin open-source do mundoO mais recente dos meus problemas é com a biblioteca (ou "gem") Paperclip, e especificamente com um programa que ela usa internamente: o Image Magick.

Aqui vou ensinar rapidamente como instalar o Image Magick no Ubuntu, o que vai resolver todos os seus problemas. 

1º Passo: Abra o seu "Terminal" e digite os seguintes comandos, em ordem: 

sudo apt-get install aptitude  
sudo aptitude install libperl-dev  
sudo aptitude install libmagickwand-dev  
sudo aptitude install libmagickcore-dev libmagick++-dev

2º Passo: Após rodar todos esses comandos no Terminal, faça o download do Image Magick mais recente clicando aqui

3º Passo: Vá ao Terminal e navegue até a pasta que você baixou o Image Magick. Chegando nela, digite os seguintes comandos, em ordem: 

tar xvzf ImageMagick.tar.gz  
cd ImageMagick-6.8.8  
./configure  
make  
sudo make install  
sudo ldconfig /usr/local/lib

Pronto, você acabou de instalar o Image Magick no Ubuntu! Para saber aonde ele foi instalado, digite o seguinte comando no Terminal: 

which convert

O console irá imprimir na tela o local em que o Image Magick foi instalado acrescido do "/convert" no final.
