[TUTORIAL] Como desinstalar / remover pacotes .RPM no Fedora?

16 November 2014 on Software Livre, Tutoriais

Você é um usuário novo no Fedora ou apenas esqueceu de como se removem pacotes RPM do seu sistema? Então você veio ao local certo: vamos aprender a desinstalar pacotes RPM do Fedora rapidamente. 

Antes de mais nada precisamos saber o nome completo do pacote que queremos remover. Para isso digite o seguinte comando no Terminal (CTRL+ALT+T): 

rpm -qa | grep -i nomeDoPrograma

Isso te retornará todos os pacotes que tem esse nome. Para desinstalar ou remover um pacote RPM em específico basta digitar: 

rpm -e nomeCompletoDoPacote

Exemplo real:
Quero remover o Mozilla Firefox, mas não sei o nome do pacote completo. Pesquiso por ele: 

[paladini@pet01 ~]$ rpm -qa | grep -i firefox  
firefox-33.0-1.fc20.x86_64

Agora que sei o nome do pacote, já posso deletar: 

[paladini@pet01 ~]$ rpm -e firefox-33.0-1.fc20.x86_64

E pronto, terminamos o nosso tutorial! Bem simples, não? Quaisquer dúvidas, podem utilizar os comentários.
