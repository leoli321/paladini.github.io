[Tutorial] Como ativar o clique no botão do meio (scroll) do mouse no Fedora / Ubuntu / Gnome

30 July 2014 on Software Livre, Dicas, Tutoriais, Fedora, Ubuntu

Você é um dos que está totalmente acostumado a utilizar o "clique no botão do meio do mouse" para algumas tarefas simples, como abrir links em outras abas, colar conteúdo copiado e fechar abas do navegador?  Se sim, conhece o grande problema que é não ter essa ferramenta disponível no seu dia-a-dia, o custo que é para se adaptar. Se você não consegue utilizar o "clique no scroll do mouse" no seu Fedora, Ubuntu ou qualquer outra DISTRO que tenha como interface o Gnome, saiba que aqui você vai encontrar a solução.
Depois de muito pesquisar na Internet, achei uma solução simples ativar o clique no botão do meio do mouse.  Abra o seu Terminal (Ctrl + Alt + T) e digite o seguinte comando: 

gsettings set org.gnome.settings-daemon.peripherals.mouse middle-button-enabled true

Espero que isso tenha solucionado o seu problema, caso não, sinta-se à vontade para perguntar nos comentários. Se te ajudou, também deixe um comentário de incentivo :)
Obrigado!
