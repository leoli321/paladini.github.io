[DICA] Como desativar o 'Visualizada às' no Facebook

16 October 2014 on Dicas

Um recurso que muita gente detestou quando o Facebook lançou foi o "Visualizada às", um pequeno texto no canto inferior do chat do Facebook que indica que determinada mensagem já foi vista pelo seu contato. 

Essa mensagem é prática quando se quer saber se determinada pessoa já leu uma mensagem e não respondeu, entretanto isso nem sempre é algo desejável. Bem, pelo menos não é desejável quando você não quer que saibam que você já leu uma mensagem e não respondeu.

Para esses casos existe uma extensão para o Mozilla Firefox / Google Chrome que "desabilita" esse recurso do Facebook. (Existe uma descrição mais técnica para quem quiser, mas não é muito difícil de imaginar o que essa extensão faz se você for desenvolvedor web).

Para baixar essa extensão salvadora chamada "Chat Undetected", clique aqui. Dúvidas podem ser tiradas nos comentários logo abaixo.
