[Tutorial] Como instalar impressoras HP no Ubuntu / Linux

17 November 2013 on Software Livre, Tutoriais

Você tem uma impressora HP e está tendo problemas para conectá-la com seu computador que roda algum sistema operacional baseado no Linux? Neste pequeno tutorial você vai aprender a instalar o "HP Device Manager" no seu Ubuntu, Linux Mint, Elementary OS, Fedora, SUSE, Debian e outras distribuições Linux. Vamos usar o HPLIP OpenSource

Seguindo este tutorial você conseguirá instalar os drivers da sua impressora HP e rodar o programa da sua impressora (conhecido como HPLIP) em praticamente qualquer distribuição Linux. 

Antes de mais nada, verifique se a sua impressora

1º Passo: Baixe os arquivos necessários:

cd ~/Downloads && wget http://prdownloads.sourceforge.net/hplip/hplip-3.13.11.run

2º Passo: Execute o arquivo:

sudo sh ~/Downloads/hplip*.run

Agora siga o guia da instalação e vá respondendo as perguntas necessárias. A instalação é muito simples, basta seguir todas as instruções - mas caso você tenha dúvidas, sinta-se a vontade para perguntar.
