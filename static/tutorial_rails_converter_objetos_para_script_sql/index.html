[Tutorial - Rails] Converter objetos para Script SQL

01 February 2014 on Software Livre, Dicas, Projetos, Rails

Esses dias estava com um problema pois queria transformar objetos Ruby em scripts de inserção SQL. Sim, aqueles famosos "INSERT INTO ...". Encontrei uma "gem" chamada "models-to-sql", mas para minha infelicidade ela não suportava Ruby on Rails 4. 

Entrei em contato com o autor e este não se mostrou interessado em continuar dando suporte a gem. Como eu realmente precisava dela, dei um fork em seu projeto e descobri como portar ela para Rails 4. Mudei seu nome para "models-to-sql-rails" e você pode encontrar o repositório no Github

Com o "models-to-sql-rails" você consegue facilmente converter objetos e array de objetos Ruby em scripts de inserção de banco de dados (em SQL). Se você precisa disso e quer uma solução fácil eu recomendo: baixe agora o "models-to-sql-rails"
